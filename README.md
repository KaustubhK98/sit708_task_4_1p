# Event Planner App

# Description

This Android app allows users to create, view, update, and delete events such as work meetings, social plans, travel plans, and appointments.

## Features
- Add events with details like event title, category, location, and date/time
- View upcoming events sorted by date
- Edit & delete existing events
- Local data persistence using Room Database
- Fragment-based navigation using Jetpack Navigation Component
- Bottom Navigation Bar
- Validation for empty title/date
- Prevention of past date selection
- Toast messages feedback for user actions


## Tech Stack
- Java
- Android Studio
- Room Database
- Jetpack Navigation Component
- RecyclerView


# Installation Instructions

Currently the app is only available on Android phones. Just install the apk from the files and run the app, no special permissions needed.
